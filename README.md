# wx-cnode
微信小程序版的cnode，基于京东凹凸实验室的Taro开发




### 安装

项目地址：（`git clone`）

```shell
git clone https://github.com/aushion/wx-cnode.git
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务

```
npm run dev:weapp
```

发布代码
```
npm run build
```
